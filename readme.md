## Стек технологий:
* препроцессор Sass в связке с GULP
* Чистый js с подключением через window.onload

## Запуск
* Заходим в фолдер
* ```npm i```
* ```npm start```
* сайт должен быть расположен по адресу ```localhost:3333```
